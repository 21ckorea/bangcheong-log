# 방청로그 (Bangcheong-Log) MVP 워크스루

## 개요
우리는 10~20대 세대가 TV 방청 경험을 모아보고 기록할 수 있는 웹 서비스인 **방청로그(Bangcheong-Log)**의 MVP(최소 기능 제품)를 성공적으로 구현했습니다.

## 핵심 구현 내용

### 1. 트렌디한 디자인 시스템 (1020세대 바이브)
- **비주얼**: "딥 다크(Deep Dark)" 배경에 생동감 넘치는 네온 악센트(바이올렛/푸크시아 그라데이션)를 사용하여 프리미엄하고 몰입감 있는 느낌을 조성했습니다.
- **글래스모피즘**: 카드와 내비게이션에 `glass` 유틸리티를 적용하여 현대적이고 층이 있는 룩을 구현했습니다.
- **마이크로 인터랙션**: `Framer Motion`을 사용하여 다음과 같은 효과를 주었습니다:
  - 페이지 진입 애니메이션 (페이드 업).
  - 버튼 호버 효과 (스케일 업).
  - 활성 내비게이션 상태 (배경 슬라이딩).

### 2. 모바일 우선 (Mobile-First) 레이아웃
데스크톱에서도 실제 앱을 사용하는 듯한 경험을 제공합니다:
- **`MobileWrapper`**: 레이아웃을 모바일 너비(`max-w-[480px]`)로 제한하고 중앙에 배치했습니다.
- **`BottomNav`**: 직관적인 아이콘이 있는 플로팅 하단 내비게이션 바를 고정했습니다.
- **`Header`**: 브랜딩이 포함된 상단 고정 헤더를 적용했습니다.

### 3. PWA (프로그레시브 웹 앱) 준비
- **설치 가능성**: `manifest.json`을 추가하여 앱처럼 설치할 수 있게 했습니다.
- **메타데이터**: 모바일 최적화를 위해 뷰포트 메타데이터(줌 방지, 테마 색상)를 설정했습니다.

### 4. 컴포넌트
- **`Button`**: 그라데이션 변형이 가능한 커스텀 애니메이션 버튼.
- **`Card`**: 방청 정보를 표시하기 위한 유리 효과 카드.
- **`Badge`**: 프로그램 카테고리 표시를 위한 알약 모양 태그.

## 검증 결과
### 자동화 테스트
- **빌드**: `npm run build`가 Next.js 14 (Turbopack) 환경에서 성공적으로 통과했습니다.
- **TypeScript**: 타입 체크를 통과했습니다 (Framer Motion `children` 타입 이슈 해결).

### 사용자 흐름 (User Flow)
1. **홈 (Home)**: 사용자는 "캐치프레이즈" 히어로 섹션과 'D-Day' 배지가 달린 현재 신청 가능한 방청 목록을 확인합니다.
2. **응모 (Apply)**: 내비게이션을 통해 응모 가이드 페이지(플레이스홀더)로 이동합니다.
3. **로그 (Log)**: 내비게이션을 통해 다이어리 페이지(플레이스홀더)로 이동합니다.
